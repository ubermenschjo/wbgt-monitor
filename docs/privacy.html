プライバシーポリシー

WBGT Monitor

対象アプリケーション

  • 事業者版WBGT記録
  • 一般版暑さアラート

開発者: STAGEN

1. 個人情報の取得

本アプリは、WBGT（暑さ指数）を取得する目的で、端末の位置情報を利用します。

  • 位置情報はWBGT値の取得のみに使用されます
  • 位置情報の処理は端末内で完結します
  • 位置情報を外部サーバーに送信・保存することはありません
※ 位置座標はOpen-Meteo APIへの天気データリクエスト時にのみ送信されます（下記「外部通信」参照）。それ以外の目的での外部送信は一切行いません。

2. データの保存

本アプリで記録されるすべてのデータは、端末内のみに保存されます。

  • 保存形式: SQLite データベース
  • 外部サーバーへのデータ同期・バックアップは行いません
  • アプリを削除すると、保存データもすべて削除されます

3. 外部通信

本アプリは以下の外部サービスと通信を行います。

  • Open-Meteo API — 天気データの取得のため、位置座標（緯度・経度）を送信します
  • 環境省 暑さ指数サイト — 暑さ指数の参照データを取得します

上記以外の外部サービスへの通信は行いません。

4. 第三者提供

本アプリは、ユーザーの個人情報を第三者に提供することは一切ありません。

5. アナリティクス・広告

本アプリは、アクセス解析ツール（アナリティクス）および広告SDKを一切使用していません。

6. プライバシーポリシーの改定

本ポリシーを改定する場合は、GitHub上で事前に告知いたします。

7. 施行日

本プライバシーポリシーは 2026年6月6日 より施行します。
