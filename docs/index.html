WBGT Monitor

暑さ指数モニター

リアルタイムWBGT（暑さ指数）を常に確認。熱中症リスクを可視化し、作業・活動の安全を守ります。事業者の労安法対応から日常の熱中症予防まで。

アカウント不要 完全オフライン対応 iOS / Android

主な機能

暑さ対策に必要なすべてを、ひとつのアプリで。

📊

リアルタイムWBGT表示

Open-Meteo APIと環境省の暑さ指数データを統合。現在地のWBGT値をリアルタイムで表示します。

⏱️

作業記録・活動記録

開始/停止ボタンで作業時間を記録。一時停止・再開にも対応。WBGT値と紐づけて履歴を保存します。

🚨

WBGT警告アラート

設定した閾値を超えるとプッシュ通知でお知らせ。危険な暑さを見逃しません。

📈

24時間予報グラフ

今後24時間のWBGT推移をグラフで確認。作業計画や外出判断に活用できます。

🔒

完全オフライン

アカウント登録不要。全データは端末内にのみ保存。プライバシーを完全に保護します。

📋

労安法対応

事業者向けは労働安全衛生法に準拠。WBGT記録の3年保存義務に対応したデータ管理機能を搭載。
